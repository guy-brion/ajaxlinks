AjaxLinks.js | jQuery plugin

This plugin allows you to ajaxify your website.
By applying this plugin to your navigation links you can load content prom other pages into the current page with one line of code!

Examples

1. Simple example


$('#nav a').ajaxlinks({
	load_to: '#content',
	load_from: '#content'
});
Page 1 | Page 2 | Page 3 (AjaxLinks.js applied)
Page 1 | Page 2 | Page 3 (Same HTML, AjaxLinks.js NOT applied)
Content will be loaded here...

2. Advanced example (with Sammy.js and a callback)


$('#nav1 a').ajaxlinks({
	load_to: '#content1',
	load_from: '#content',
	loader: 'Custom loading...',
	use_sammy: true,					
	callback: function(params) {
		$('#content1').css('opacity', '0').animate({'opacity': '1'}, 'slow');
	}
});
Page 1 | Page 2 | Page 3 (AjaxLinks.js applied)
Content will be loaded here...
